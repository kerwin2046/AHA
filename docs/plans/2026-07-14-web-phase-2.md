# Web Dashboard Phase 2 Implementation Plan

**Goal:** Add full-text search (FTS5), real-time daemon feed via SSE, and statistics charts to the web dashboard.

**Architecture:**
- Backend: SQLite FTS5 virtual table for ranked full-text search; axum SSE endpoint polls DB for new entries; new daily trends API for chart data.
- Frontend: Pure SVG charts (no new npm deps) — bar chart for top words, donut for provider breakdown, area/bar for daily trends; EventSource for live feed.
- Daemon unchanged — SSE polls the shared SQLite DB for new rows every ~2s.

**Tech Stack:** rusqlite FTS5, axum 0.7 SSE + `futures-util`, Preact + inline SVG

```rust
pub struct DailyTrend {
    pub date: String,

pub fn query_daily_trend(days: usize) -> Result<Vec<DailyTrend>> {
    let conn = init()?;
    let mut stmt = conn.prepare(
        "SELECT date(created_at) as day, COUNT(*) as cnt
         FROM queries
         WHERE created_at >= date('now', ?1)
         GROUP BY day ORDER BY day ASC"
    )?;
    let rows = stmt.query_map(params![format!("-{} days", days)], |r| {
        Ok(DailyTrend { date: r.get(0)?, count: r.get(1)? })
    })?;
    let mut trends: Vec<DailyTrend> = rows.filter_map(|r| r.ok()).collect();
    // Fill in missing days with 0
    if let (Some(first), Some(last)) = (trends.first().map(|t| &t.date), trends.last().map(|t| &t.date)) {
        // Walk from first date to today, fill gaps
    }
    Ok(trends)
}
```

### Task 3: SSE Live Feed

**Files:**

**Cargo.toml additions:** `futures-util = "0.3"`

**web.rs changes:**

1. Update `AppState` to hold a `watch::Sender<i64>`:
```rust
use tokio::sync::watch;
struct AppStateInner {
    max_id_tx: watch::Sender<i64>,
}
type AppState = Arc<AppStateInner>;
```

2. In `serve()`, spawn background task + pass state:
```rust
let (max_id_tx, _) = watch::channel(0i64);
let state = Arc::new(AppStateInner { max_id_tx: max_id_tx.clone() });

// Background poller
tokio::spawn(async move {
    let mut interval = tokio::time::interval(Duration::from_secs(2));
    loop {
        interval.tick().await;
        if let Ok(Some(max_id)) = crate::history::query_max_id() {
            let _ = max_id_tx.send_if_modified(|current| {
                if max_id > *current { *current = max_id; true } else { false }
            });
        }
    }
});
```

3. Add `query_max_id()` to `history.rs`:
```rust
pub fn query_max_id() -> Result<Option<i64>> {
    let conn = init()?;
    conn.query_row("SELECT MAX(id) FROM queries", [], |r| r.get(0))
}
```

4. Add SSE endpoint:
```rust
use axum::response::sse::{Event, Sse, KeepAlive};
use futures_util::stream::{Stream, StreamExt};
use std::convert::Infallible;

async fn api_events(
    State(state): State<AppState>,
) -> Sse<impl Stream<Item = Result<Event, Infallible>>> {
    let rx = state.max_id_tx.subscribe();
    let stream = futures_util::stream::unfold(
        (rx, 0i64),
        |(mut rx, mut last_id)| async move {
            loop {
                if rx.changed().await.is_err() {
                    return None;
                }
                let current_max = *rx.borrow();
                if current_max > last_id {
                    break;
                }
            }
            let current_max = *rx.borrow();
            let entries = crate::history::list_queries_since(last_id, 50).unwrap_or_default();
            let new_last = entries.last().map(|e| e.id).unwrap_or(last_id);
            let mut events: Vec<Result<Event, Infallible>> = entries.into_iter().map(|e| {
                Ok(Event::default().json_data(e).unwrap_or_default())
            }).collect();
            if events.is_empty() {
                events.push(Ok(Event::default().data("ping")));
            }
            Some((futures_util::stream::iter(events), (rx, new_last)))
        },
    ).flatten();
    Sse::new(stream).keep_alive(KeepAlive::default())
}
```

Register: `.route("/api/events", get(api_events))`
